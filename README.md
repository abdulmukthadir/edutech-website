#Eduvi Landing Page (Figma to Code)

A modern landing page UI built using HTML and Tailwind CSS, converted from a Figma design.

#Overview

This project is a pixel-perfect implementation of a course platform landing page. It focuses on clean UI design, proper layout structure, and modern styling techniques.

The page includes multiple sections such as hero, courses, categories, testimonials, and footer.

#Features
Pixel-perfect Figma to code conversion
Gradient-based modern UI
Structured and reusable layout
Hover effects and transitions
Clean and readable code
Sections Included
Hero Section
Search Bar
Featured Courses
Top Categories
